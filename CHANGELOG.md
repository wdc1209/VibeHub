# Changelog

## 1.0.0 - 2026-04-02

First public macOS release of Vibe Hub.

- Renamed the project from `Token Card` to `Vibe Hub`
- Shipped native macOS app bundle `VibeHub.app`
- Added compact mode and full mode UI flows
- Added voice capture, rewrite, compress, and multi-target send flows
- Added GitHub release packaging via `vibe-hub-mac/package-release.sh`
- Added branded app icon and in-app logo
- Added white-background app icon source pipeline
- Added release docs, privacy note, and notarization scaffold
- Future releases will expand support for more software targets and add Windows distribution
