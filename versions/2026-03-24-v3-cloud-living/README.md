# Vibe Hub · V3 Cloud Living

这是沿 **B 方向 / 云朵生命感版** 打磨的一稿。

## 这一版的收束目标
- 更柔软轮廓
- 更像呼吸的漂浮
- 更弱工具感
- 更像一团悬浮的语言云

## 相比 V2 的变化
- 去掉了三种外形切换，避免分叉，直接收敛到单一方向
- 主卡片改成更轻、更透明的玻璃悬浮稿，并重新拉回上下漂浮感
- 顶部信息收缩：左上角仅保留小号 `Vibe Hub`
- 顶部与历史抽屉文案改回更直白的产品表达（发送目标 / 发送历史 / 原始输入）
- 去掉两行过于概念化的说明文案
- 保留了拖动、编辑、重写、压缩、发送、历史抽屉这些核心交互

## 关于桌面截图背景
原计划把网页背景直接替换成当前桌面截图，用于观察真实桌面上的悬浮效果。
但当前运行环境里无法直接调用系统截图命令，所以这一步暂时没有自动完成。

如果补入一张桌面截图，可以直接替换背景继续看真实效果。

## 现在主要在验证什么
1. 云团轮廓是否已经足够脱离“普通玻璃卡片”
2. 漂浮感是否更接近“有呼吸的语言块”
3. 工具感是否被削弱到合适程度
4. 历史栈和发送动作在更软的视觉里是否仍清楚
