# Vibe Hub 1.0.0

First public macOS release of Vibe Hub.

## Highlights

- Native macOS app bundle
- Full mode and compact mode layouts
- Voice input with speech recognition
- Rewrite and compress actions
- Multi-target send flow
- Clipboard target
- Codex, AntiGravity, and OpenClaw Web Chat integration paths
- Updated Vibe Hub branding and app icon

## Install

1. Download `VibeHub-macOS-1.0.0.zip`
2. Unzip it
3. Drag `VibeHub.app` into `Applications`
4. Open the app from Applications or Launchpad

## Notes

- Current public build is ad-hoc signed
- Apple notarization is not included in this archive
- Microphone and Speech Recognition permissions are required for voice input
- API keys and local memory are not included in the release package
- Users should configure cloud speech / LLM keys locally and must not commit them to git
- It is acceptable to use another AI tool locally to help write config files, but real keys must remain local
- Future releases will gradually add support for more software and apps
- Windows distribution is planned for a later release
